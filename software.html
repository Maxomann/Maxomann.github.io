Island Survival

Language: C++

Libraries: SFML

Island Survival is a small tilemap based sandbox game.
You can create your own worlds out of tiles and objects (Trees, rocks, etc.).
More over you can add your own objects to the game.
To do so you have to write an object file (.object), save it in the 'Objekte' folder
and copy the corresponding texture to the 'Texturen' folder.

Tilemap Netedit

Language: C++

Libraries: SFML, TGUI, RapidXml

Tilemap Netedit is a multiplayer (Server/Client) tilemap editor.
You can edit an infinitely large tilemap with your friends in real time.
To connect to your friends game you have to choose 'Connect to server' in the menu.
Then you have to type in your friends Ip Address and a port.
You can choose any port as long as it is not used by another application and forwarded to your friends pc in his local network.

Download - Tilemap Netedit (Windows)
